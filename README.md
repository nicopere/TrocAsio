# Gestion de stock

Ce projet logiciel est une spécialisation de l'application web développée par Mohammed El Badry pour la gestion de stock d'une entreprise.

Il s'agit ici de faciliter la gestion d'un projet pédagogique d'achat/vente de calculatrices d'occasion au [lycée Dupuy de Lôme](https://lyceedupuydelomebrest.fr/).

## Prérequis

Pour tester ce projet, vous aurez besoin d'un server LAMP/WAMP:

- Linux (ou Windows }-]);
- Apache (par exemple);
- MySQL (ou MariaDB);
- PHP.

Et bien sûr d'un navigateur web (Firefox entre autres).

## Spécialisation

Il est possible le projet soit applicable tel quel! Même si dans notre cas la table des fournisseurs (resp. des clients) comportera sans doute autant de lignes que celles des approvisionnements (resp. des commmandes).

Il y aura peut-être quelques ajustements pour l'affichage sur un _smartphone_ mais ce n'est même pas sûr (Bootstrap).
